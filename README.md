# Bank Marketing Analysis

This project analyzes customer behavior and campaign effectiveness in bank marketing campaigns.

## Objective
- Identify key factors influencing customer subscription
- Evaluate campaign effectiveness
- Provide data-driven recommendations

## Key Insights
- Conversion rate: 14.6%
- Longer call duration increases conversion
- Optimal contact frequency: 1–3 times
- Contact type significantly influences outcomes

## Tools
- Python (Pandas, Seaborn)
- Google Colab
